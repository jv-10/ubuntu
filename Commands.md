# Commands
Ubuntu useful commands.


## Install Download files from Terminal

1. Navigate to folder of download 

```
cd Downloads
```

2. dpkg

```
sudo dpkg -i <deb.deb>
```

3. install

```
sudo apt-get install -f
```

## sudo in file explorer

Execute nautilus

```
gksu nautilus
```

## Change file in gedit as sudo

```
sudo -i gedit <path>
```
